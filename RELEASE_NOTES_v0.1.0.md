# 픽셀부스터 v0.1.0 🎉

더카당 픽셀부스터의 첫 번째 정식 릴리즈입니다!

## ✨ 주요 기능

### 이미지 변환
- ✅ WebP, AVIF, JPG, PNG 포맷 지원
- ✅ 배치 처리로 여러 파일 동시 변환
- ✅ 품질 조절 가능 (1-100)
- ✅ 원본 파일 자동 백업

### 사용자 경험
- ✅ 직관적인 드래그 앤 드롭 인터페이스
- ✅ 실시간 진행 상태 표시
- ✅ Excel 기반 작업 로그
- ✅ 다국어 지원 (한국어/영어)

### 시스템
- ✅ 자동 업데이트 기능
- ✅ 구독 등급별 기능 제한
- ✅ 기기 인증 시스템

## 🐛 버그 수정

### Phase 6 안정화
- 🔧 Preload script window 객체 접근 오류 수정
- 🔧 Excel 로그 기록 시 컬럼 인덱스 오류 수정
- 🔧 로그인/회원가입/이메일 인증 개선
- 🔧 RLS 정책 오류 수정

## 📦 설치 방법

1. `픽셀부스터 Setup 0.1.0.exe` 다운로드
2. 설치 파일 실행
3. 안내에 따라 설치 진행

## 🔄 자동 업데이트

설치 후 자동 업데이트가 활성화됩니다. 새 버전이 출시되면 자동으로 알림을 받고 업데이트할 수 있습니다.

## 📝 시스템 요구사항

- Windows 10 이상 (x64)
- 4GB RAM 이상 권장
- 100MB 이상의 여유 공간

## 🙏 감사합니다

더카당 픽셀부스터를 사용해주셔서 감사합니다!
