# 픽셀부스터 v0.1.0 - 수동 테스트 가이드

**테스트 대상**: 픽셀부스터 Setup 0.1.0.exe
**예상 소요 시간**: 1-2시간
**필수 환경**: Windows 10/11 (64-bit)

---

## 🚀 테스트 시작 전 준비

### 1. 빌드 파일 위치
```
F:\The kadang\code_project\claude\thekadang_pixelbooster\dist-electron\픽셀부스터 Setup 0.1.0.exe
```

### 2. 테스트 이미지 파일 준비 (권장)
다음 형식의 이미지 파일을 5-10개 준비하세요:
- **JPG**: 1-5MB (5개)
- **PNG**: 1-5MB (5개)
- **GIF**: 100KB-1MB (선택 사항)
- **TIFF**: 5-10MB (선택 사항)

### 3. 테스트 계정 준비
- **이메일**: 테스트용 유효한 이메일 주소
- **비밀번호**: 최소 6자 이상

## 📋 테스트 체크리스트

### Phase 1: 설치 및 첫 실행 (10분)

#### 1.1 설치 파일 실행

- [ ] **4단계**: "설치" 버튼 클릭
- [ ] **5단계**: 설치 진행 바 확인 (0-100%)
- [ ] **6단계**: 설치 완료 메시지 확인

**예상 결과**:
- ✅ 에러 없이 설치 완료

**오류 발생 시**:

#### 1.2 첫 실행

### Phase 2: 인증 시스템 (15분)

- [ ] **1단계**: "회원가입" 버튼 클릭
- [ ] **2단계**: 이메일 입력 (예: test@example.com)

- [ ] **4단계**: 비밀번호 확인 (동일하게 입력)
- [ ] **5단계**: "회원가입" 버튼 클릭
- [ ] **6단계**: 성공 메시지 확인

**예상 결과**:

- ✅ 로그인 화면으로 자동 이동

**에러 테스트**:
- [ ] **중복 이메일**: 동일한 이메일로 다시 회원가입 시도 → "이미 사용 중인 이메일입니다" 에러
- [ ] **잘못된 이메일**: "test@" 입력 → "유효하지 않은 이메일 형식입니다" 에러
- [ ] **비밀번호 불일치**: 확인란에 다른 비밀번호 입력 → "비밀번호가 일치하지 않습니다" 에러

#### 2.2 로그인

  - [ ] WebP만 선택 가능 확인

  - [ ] 배치 크기 5개 제한 (6개 이상 추가 시 경고)

**예상 결과**:
- ✅ Free 등급: WebP만 활성화, 나머지 비활성화
- ✅ 제한 초과 시 에러 메시지 표시
- ✅ "Pro로 업그레이드" 버튼 표시

#### 3.5 출력 폴더 선택 및 변환 실행
- [ ] **1단계**: "출력 폴더 선택" 버튼 클릭
- [ ] **2단계**: 폴더 선택 다이얼로그에서 출력 경로 선택 (예: 바탕화면\output)
- [ ] **3단계**: "변환 시작" 버튼 클릭

- [ ] **6단계**: 변환 완료 알림 확인

**예상 결과**:

- ✅ 병렬 처리 (최대 4개 동시)
- ✅ 변환 완료 알림: "10개 파일 변환 완료"
- ✅ 출력 폴더에 .webp 파일 10개 생성
- ✅ 파일 크기 30-50% 감소 확인

#### 3.6 성능 측정
- [ ] **처리 속도 확인**:
  - 1MB 이미지: < 3초
  - 5MB 이미지: < 10초
- [ ] **압축률 확인**:
  - 원본 10MB → 변환 후 5MB (약 50% 압축)

- [ ] **2단계**: 동일 파일 10회 반복 변환
- [ ] **3단계**: 앱 크래시 여부 확인
- [ ] **4단계**: 메모리 누수 여부 확인 (메모리 사용량 증가 없어야 함)

**예상 결과**:
- ✅ 앱 크래시 없음
- ✅ 메모리 누수 없음
- ✅ 10회 반복 변환 성공
